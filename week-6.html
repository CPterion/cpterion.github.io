Week #6: Robot Framework Integration

Steven Jethro Vitangcul

Sat 19 July 2025 • 1 min read

After learning how to use Robot Framework, we incorporated our codes into it. I also reworked on some functions so that it’s usable as robot tests. I added a new functionality for the previous week’s code: for each process IDs, I applied the terminate and suspend checking to them. Good thing I made separate functions for those, so it was easy to do. We finally merged our codes, and exporting each feature was easy (though the coding styles were very different). After that, we wrote robot tests for each feature that we made. I’m not too sure about this, but I think I encountered a bug with Robot Framework’s regex library. I had no idea why it didn’t work, but I just changed the print statements so that it includes the PID and now the regex is not needed anymore. We also shared a document so that we can both work on the documentation for the features we made. It included the usage, limitations, and unfinished features. I also made flowchart representations of the codes I wrote. Last Friday however, due to bad weather, I had to stay and work from home. I hope the weather stabilizes next week.
