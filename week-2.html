Week #2: Task Automation Software Exploration

Steven Jethro Vitangcul

Sat 21 June 2025 • 1 min read

For our second week, we got into using GitLab as it is the website where we can access the software we’ll be working on. We learned how to use the Company Portal to install the programs that were reviewed by the Security Team. They gave us a task to configure Disabler and make it work on our device. I encountered an error (which is still not resolved until now) but it didn’t affect the execution of the software, so I was able to ignore it. We were also assigned to look for task automation applications, either coded or through recording macros. I researched and found Power Automate but it was too slow, both in running and recording macros (probably because it runs in the cloud) so it was not chosen. AutoHotkey was another candidate but was unfortunately not in the Company Portal so I wasn’t able to test it fully but was able to run it through a Python binding. It was promising but it still wasn’t chosen. There was another library in Python which is RPA Framework that I thought could be used but I wasn’t able to make it work. I continued the New Employee Onboarding after for the rest of the week.
